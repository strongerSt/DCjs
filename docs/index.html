代码解密工具 是一个开源工具，用于解密各种混淆的代码。

支持的解密类型:

  • Eval混淆 - 嵌套的eval函数混淆
  • Obfuscator - 使用JavaScript-Obfuscator等工具混淆的代码
  • JJEncode - 使用JJEncode混淆的代码
  • AAEncode - 使用AAEncode (ﾟДﾟ) 混淆的代码
  • JSFuck - 使用[]()!+字符混淆的代码
  • Base64 - Base64编码的内容
  • URL编码 - URL编码的内容
  • 十六进制 - 十六进制表示的内容

全新的客户端解密实现，所有解密操作在本地完成，无需向服务器发送请求，更安全、更高效。

注意事项:

  • 本工具仅供学习和研究使用，请勿用于非法用途。
  • 解密操作完全在您的浏览器中进行，代码不会上传到任何服务器。
  • 对于复杂的混淆可能无法完全解密，但会尽可能提供最佳结果。
  • 如遇到无法解密的情况，您可以尝试选择不同的解密类型。
