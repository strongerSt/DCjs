关于 Mikephie 脚本解密工具 (浏览器版)

这是一个用于解密各种JavaScript混淆代码的在线工具。它直接在浏览器中处理解密，无需服务器支持。

支持的解密类型：

  • Common - 通用混淆解密
  • JJencode - JJencode混淆解密
  • Sojson - Sojson混淆解密
  • SojsonV7 - Sojson V7混淆解密
  • Obfuscator - JavaScript Obfuscator混淆解密
  • Awsc - 阿里云混淆解密
  • Part2AI - 部分AI混淆解密

工作原理：

1. 用户提交混淆代码

2. 浏览器直接使用内置的解密库处理代码

3. 解密结果立即显示在页面上

4. 所有处理均在本地完成，不会发送到任何服务器

浏览器版与服务器版区别：

- 浏览器版: 所有解密直接在浏览器中进行，不需要GitHub令牌，但解密能力可能有限

- 服务器版: 通过GitHub Actions进行解密，可以处理更复杂的混淆，但需要创建GitHub issue

隐私说明：

本工具在浏览器版中不会将您的代码发送到任何服务器。所有处理都在您的浏览器中本地完成，确保您的代码安全和隐私。

局限性：

由于浏览器环境的限制，某些高级混淆可能无法完全解密。如果浏览器版无法满足您的需求，请考虑使用GitHub版本。
